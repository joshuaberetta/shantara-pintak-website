# Shantara Pintak - Personal Website

A clean, professional personal website showcasing education, experience, and areas of focus in humanitarian action and peacebuilding.

## Deploying to GitHub Pages

1. **Create a new repository** on GitHub named `shantara-pintak.github.io` (or any name you prefer)

2. **Upload these files** to the repository:
   - `index.html`
   - `Shantara_Pintak_Resume.pdf`

3. **Enable GitHub Pages**:
   - Go to your repository's **Settings**
   - Navigate to **Pages** in the left sidebar
   - Under "Source", select **Deploy from a branch**
   - Choose the **main** branch and **/ (root)** folder
   - Click **Save**

4. Your site will be live at `https://YOUR-USERNAME.github.io/REPO-NAME/` within a few minutes!

## Customization

- Edit `index.html` to update content
- Replace `Shantara_Pintak_Resume.pdf` with an updated resume (keep the same filename)
- Colors can be changed in the CSS `:root` variables at the top of the `<style>` section
